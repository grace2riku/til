# 20240827
所有権の変更コマンドについて学んだ。

ファイルやディレクトリの所有者を変更する方法

参考: https://qiita.com/3062_in_zamud/items/d93770016e7d931a5983

chown -R user:group dir

* -R	ディレクトリ内の所有権も変更する
* user: 変更したいユーザー名
* group: 変更したいグループ名
* dir: 所有者を変更したいディレクトリ
